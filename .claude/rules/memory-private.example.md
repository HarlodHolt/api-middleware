# Private Memory (Example)

## Purpose

Template for local-only notes that should not be committed.

Copy this file to:

- `.claude/rules/memory-private.md`

That file is gitignored by `.claude/.gitignore`.

## Good Candidates

- Personal contact details
- Private credentials locations (never actual secrets)
- Non-public customer or business context
- Sensitive reminders you want loaded locally

## Do Not Store

- Raw secrets
- API keys
- Passwords
- Tokens
